Place your media files here:

- gameplay-clip.mp4  — Screen recording of a good gameplay run (vertical 1080x1920 preferred)
- track.mp3          — Background music track

Tips for recording gameplay:
1. On iPhone: Settings → Control Center → add Screen Recording. Record a run in portrait.
2. On Android: Most phones have built-in screen record in the notification shade.
3. On desktop: Use OBS Studio set to 1080x1920 canvas, record from your browser.

The files should be placed directly in this /public folder so Remotion can
access them via staticFile().
