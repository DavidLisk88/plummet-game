📖 Plummet Notification Icon

Word of the Day notification icon for Android

Installation Instructions

  1. Click "Download All Icons" or download individually
  2. Place icons in the Android project:
    • android/app/src/main/res/drawable-mdpi/ic_notification.png (24px)
    • android/app/src/main/res/drawable-hdpi/ic_notification.png (36px)
    • android/app/src/main/res/drawable-xhdpi/ic_notification.png (48px)
    • android/app/src/main/res/drawable-xxhdpi/ic_notification.png (72px)
    • android/app/src/main/res/drawable-xxxhdpi/ic_notification.png (96px)
  3. Run npx cap sync android

Note: Android notification icons must be white silhouettes with transparency. The system applies color tinting automatically.
